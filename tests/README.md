# Тесты

Для каждого **нового или изменённого модуля** в `src/` нужны тесты в `tests/`.

- Структура: `tests/test_<модуль>.py` или `tests/<подсистема>/test_<файл>.py`, зеркально `src/`.
- Запуск: из корня проекта — `pytest` или `python -m pytest`.
- Модуль считаем готовым только когда тесты проходят: так проверяем, что всё подходит и работает, а не угадываем.
