# Дорожная карта (краткая рабочая версия)

Этот файл оставлен в docs как компактный список направлений.
Полная версия без сокращений перенесена в архив:

- docs/archive/ROADMAP.full.md

## Приоритеты

1. Автономность и планирование: расширять цели и контроль риска инструментов.
2. Мониторинг и производительность: tool_times, alerts, регулярный экспорт.
3. Интеграции: унификация конфигурации внешних сервисов.
4. Качество: тесты на каждый новый/изменённый модуль.

## Правило ведения

- В краткой дорожной карте в docs держим только активные приоритеты.
- Исторические детали и длинные обоснования ведём в docs/archive/ROADMAP.full.md.
