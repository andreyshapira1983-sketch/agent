# Индекс документов агента

Один файл — одна строка: путь и краткое содержание. Агент использует этот индекс, чтобы открыть нужный документ для деталей. Все перечисленные файлы сохраняются; не удаляются.

| Документ | Путь | Содержание |
|----------|------|-------------|
| **Мастер-документ** | docs/AGENT_MASTER.md | Единая ориентация: архитектура, структура проекта, цикл, эволюция, инструменты, Telegram, таблица документов. |
| **Индекс (этот файл)** | docs/AGENT_DOCS_INDEX.md | Список всех документов с путём и одним предложением. |
| Changelog и фичи | docs/CHANGELOG_AGENT_FEATURES.md | Что сделано, будет ли работать; Telegram+Local; pip, run_python/PowerShell, фото/видео, библиотеки, intent, проактивность, Planner Loop. |
| Архитектура агента | docs/AGENT_ARCHITECTURE.md | Агент первый, LLM — генератор; видимость workspace; request_patch и защитный слой. |
| README | docs/README.md | Запуск, требования, .env, команды Telegram, память, автономность, эмоции, алерты. |
| Автономия | docs/AUTONOMY.md | Цикл observe→reason→plan→act→reflect→improve; оркестратор, policy, очередь, task_creator, autonomy_tools, конфиг. |
| Эволюция | docs/EVOLUTION.md | Guardrails, sandbox, patch_guard, task_guard, evolution_lock, применение патчей, откат. |
| Семья агентов | docs/FAMILY_AGENTS.md | Семейка агентов, фантазийные мысли, генеалогия, дочерние агенты. |
| Использование | docs/USAGE.md | Как ставить задачи, как добавлять инструмент. |
| Тестирование | docs/TESTING.md | Линтер, mypy, bandit, pytest; скрипты PowerShell; артефакты, nightly. |
| Обратная связь | docs/FEEDBACK.md | Каналы обратной связи, категории, шаблон. |
| Дорожная карта | docs/ROADMAP.md | Краткая дорожная карта; полная версия в archive. |
| План архитектуры | docs/ARCHITECTURE_PLAN.md | Краткий план архитектуры; полная версия в archive. |
| Генетика | docs/GENETICS.md | Краткая генетическая модель; полная версия в archive. |
| Карта документации | docs/INDEX.md | Рабочий минимум документов и архив. |
| Видимость агента | docs/AGENT_VISIBILITY_TEST.md | Тесты видимости workspace для агента. |
| Архив: план архитектуры | docs/archive/ARCHITECTURE_PLAN_FULL.full.md | Полная версия плана архитектуры. |
| Архив: генетика | docs/archive/GENETICS.full.md | Полная версия генетической модели. |
| Архив: дорожная карта | docs/archive/ROADMAP.full.md | Полная версия дорожной карты. |

---

**Как пользоваться:** Сначала ориентироваться по **AGENT_MASTER.md**. Для углубления в тему — открыть документ из этой таблицы по пути в колонке «Путь».
