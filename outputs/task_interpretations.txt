# Возможные интерпретации задачи "Определить адресата приветствия"

1. **Анализ существующего текста**
   - Найти в тексте приветствие (Привет, Здравствуйте, Hello и т.д.)
   - Определить, к кому оно обращено (имя после приветствия)

2. **Создание системы определения адресатов**
   - Разработать алгоритм для извлечения адресатов из текстов
   - Учесть разные форматы: "Привет, Андрей!", "Дорогой друг", и т.д.

3. **Анализ переписки/email**
   - Определить получателя сообщения по заголовкам
   - Извлечь имя из поля "To:" или из текста письма

4. **Контекст взаимодействия с Андреем**
   - Возможно, нужно определить, что приветствие адресовано Андрею
   - Как партнёру по решению задач

Без дополнительного контекста выбираю вариант 4.
